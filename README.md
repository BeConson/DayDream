# DayDream

DayDream 白日梦商城，目前只适应移动端的 Demo。基于 Spring Boot + Vue 进行前后端分离开发。

- 前端：Vue全家桶
- 后端：Spring Boot + MySQL
- 工具：Intellij IDEA
